# Desafio_Zetta
  Repositorio destinado ao Desafio Prático do PS Zetta 2020

# Tecnologias utilizadas:
- Java (backend)
- HTML e CSS (frontend)
- MYSQL (banco de dados)
- Spring Boot (setup do app)

# Dependências do Spring Boot:
- Spring Boot DevTools
- Spring Web
- Spring Data JPA
- Thymeleaf

# IDEs utilizadas:
- Eclipse 
- MYSQL Workbench
